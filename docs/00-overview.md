# Homelab Overview

This document describes the sanitized public overview of the homelab.

## Goals

- Build repeatable infrastructure documentation
- Keep public examples safe and sanitized
- Document security decisions before production use
- Make rebuild and recovery possible from notes and automation

## Public Safety Rules

Do not publish real passwords, API tokens, private keys, internal IP maps, firewall exports, serial numbers, customer data, or private inventory.

## High-Level Areas

- Compute and virtualization
- Network and segmentation
- Identity and access
- Monitoring and logging
- Backup and disaster recovery
- Automation and configuration management
